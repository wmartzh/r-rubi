# r-rubi
This a sandbox project to make a food orders API

## Dependencies Required

| Dependencies      | Description | Version|
| ----------- | ----------- | ----------- |
| Prisma      | ORM       |2.23.0
|@prisma/client| ORM Client | 2.23.0
  

## Installation

    npm install

    yarn install
### Run Migrations
    
    prisma migrate dev --name init 

###  Run server

    npm run server 

    yarn server
